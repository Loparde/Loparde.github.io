Sparta Game

Jump and run game in old sparta style. You can ride spears and have to solve puzzles by doing so.
This is also my first experience with pixel art.
Multiplayer Game
This game came to life because I wanted to learn about the multiplayer plugin NETCODE in Unity.
It is intended to be game that relies on analysing the other players behaviour to identify infected people and stop a virus form spreading by putting them into quarantine.
